# Feature spec template

Use this section order in `docs/specs/<slug>.md`:

1. Summary
2. Player-facing behavior
3. Assumptions
4. Scope
5. Out of scope
6. Affected areas
7. Acceptance criteria
8. Risks / follow-ups

## Example acceptance criteria bullets
- Player can start the quest from NPC `npc_town_guide`.
- Quest requires killing exactly 10 slime-type enemies.
- Completion grants 500 gold one time per day.
- Reward dialogue appears after successful completion.
- Existing daily quest reset behavior remains unchanged.
